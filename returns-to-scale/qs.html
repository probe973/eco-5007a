For each of the following production functions, identify the returns to scale.

  1. $Q(K,L)=3K+2L$

    Show Solution

    Solution to 1

    $Q(2K,2L)=6K+4L=2(3K+2L)=2Q(K,L)$

    Doubling input doubles output, so constant returns to scale.

  2. $Q(K,L)=0.2KL$

    Show Solution

    Solution to 2

    $Q(2K,2L)=0.2(2K)(2L)=0.2 \times 2 \times 2 KL=0.8KL=4Q(K,L)$

    Doubling input more than doubles output, so increasing returns to scale.

  3. $Q(K,L)=\frac{KL}{2}$

    Show Solution

    Solution to 3

    $Q(2K,2L)=\frac{(2K)(2L)}{2}=2KL=4Q(K,L)$

    Doubling input more than doubles output, so increasing returns to scale.

  4. $Q(K,L)=K^2L^2$

    Show Solution

    Solution to 4

    $Q(2K,2L)=(2K)^2(2L)^2=(4K^2)(4L^2)=16K^2L^2=16Q(K,L)$

    Doubling input more than doubles output, so increasing returns to scale.

  5. $Q(K,L)=K^{0.5}L^{0.5}$

    Show Solution

    Solution to 5

    $Q(2K,2L)=2^{0.5}K^{0.5}2^{0.5}L^{0.5}=2^{0.5}2^{0.5}K^{0.5}L^{0.5}=2Q(K,L)$

    Doubling input doubles output, so constant returns to scale.

  6. $Q(K,L)=K^{0.9}L^{0.1}$

    Show Solution

    Solution to 6

    $Q(2K,2L)=2^{0.9}K^{0.9}2^{0.1}L^{0.1}=2Q(K,L)$

    Doubling input doubles output, so constant returns to scale.

  7. $Q(K,L)=K^{0.2}L^{0.6}$

    Show Solution

    Solution to 7

    $Q(2K,2L)=2^{0.2}K^{0.2}2^{0.6}K^{0.6}=2^{0.8}Q(K,L) \approx 1.74Q(K,L)$

    Doubling input leads to a less than doubling of output, therefore decreasing returns to scale.

  8. $Q(K,L)=K^{0.8}L^{0.3}$

    Show Solution

    Solution to 8

    $Q(2K,2L)=2^{0.8}2^{0.3}K^{0.8}L^{0.3}=2^{1.1}Q(K,L) \approx 2.14Q(K,L)$

    Doubling input more than doubles output, so increasing returns to scale.

  9. $Q(K,L)=K^{\alpha}L^{\beta}$ where

    1. $\alpha + \beta=1$

    2. $\alpha + \beta<1$

    3. $\alpha + \beta>1$

    Show Solution

    Solution to 9

    1. $\alpha+\beta=1 \implies Q(2K,2L)=2Q(K,L)$ constant

    2. $\alpha+\beta<1 \implies 2^{\alpha+\beta}<2 \implies Q(2K,2L)<2Q(K,L)$ decreasing

    3. $\alpha+\beta>1 \implies 2^{\alpha+\beta}>2 \implies Q(2K,2L)>2Q(K,L)$ increasing
